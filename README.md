# -An-lisis-del-Mercado-de-Viajes-Compartidos-en-Chicago
Proyecto de análisis de datos orientado a comprender el comportamiento del mercado de viajes compartidos en Chicago mediante el estudio de datos operativos, información de competidores y condiciones climáticas.
Se analizaron patrones de demanda, concentración del mercado por compañías, zonas con mayor volumen de viajes y el impacto del clima en la duración de los trayectos.

El proyecto incluyó consultas SQL, análisis exploratorio en Python, visualización de datos y pruebas de hipótesis estadísticas, con el objetivo de apoyar la toma de decisiones estratégicas y operativas en un contexto real de negocio.

Herramientas: SQL, Python, Pandas, Matplotlib, análisis estadístico
